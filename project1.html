Review Chatbot

This application helps users interact with restaurant reviews by analyzing and extracting insights from customer feedback in real time. It allows users to discover top-rated dishes, get personalized recommendations, and explore overall dining experiences with ease. Restaurant owners can track trends and customer satisfaction over time.

Contributions

  • Designing and developing the application architecture
  • Integrating Redpanda for real-time streaming
  • Utilizing OpenAI embeddings for extracting insights

Future Vision

This application aims to be more than just a restaurant review tool. In the future, we envision integrating more personalized features like user preferences and dietary restrictions, expanding beyond restaurants to other industries, and providing an even richer user experience with advanced data analytics.
